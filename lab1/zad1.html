Syllabus Bootcampu: Fullstack Developer

  1. Moduł Frontend
    1. Podstawy HTML
    2. Zaawansowany CSS i RWD
      • Flexbox i Grid
        1. Tworzenie layoutu
        2. Centrowanie elementów
        3. Responsywność (Mobile First)
      • Animacje CSS
      • Preprocesor SASS
    3. JavaScript Podstawy
  2. Moduł JavaScript Zaawansowany
    1. Programowanie Obiektowe
    2. Asynchroniczność (Promises, Async/Await)
      • Czym jest Event Loop
        1. Call Stack
        2. Web APIs
        3. Callback Queue
      • Metoda fetch()
      • Obsługa błędów (try/catch)
    3. Praca z DOM
  3. Moduł Backend (Node.js)
    1. Wstep do Node.js
    2. Express i Rest API
      • Routing HTTP
        1. Metoda GET
        2. Metoda POST
        3. Kody odpowiedzi
      • Middleware
      • Walidacja danych (Joi)
    3. Praca z bazą danych
  4. Moduł Bazy Danych
    1. Relacyjne BD
    2. Bazy NoSQL (MongoDB)
      • Kolekcje i dokumenty
        1. Operacje CRUD
        2. Agregacje
        3. Indeksowanie (Optymalizacja)
      • Biblioteka Mongoose
      • Relacje w NoSQL
    3. Wdrożenie projektu

Wprowadzenie do algorytmiki

Zrozumienie podstaw algorytmiki jest kluczowe dla każdego programisty. Algorytm to nic innego jak skończony ciąg jasno zdefiniowanych czynności koniecznych do wykonania pewnego zadania.
